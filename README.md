🌾 FarmProfit

FarmProfit is a Farm Expense Management System developed using FastAPI, Streamlit, and MySQL. It helps farmers record farm operations, manage expenses, and track driver payments in one place.

---

🚀 Features

👤 Farmer Management
- Farmer Registration
- Login Authentication
- Profile Management

🌾 Field Management
- Add Fields
- Update Fields
- Delete Fields
- Total Land Calculation

🚜 Plowing Module
- Record Plowing Details
- Driver Selection
- Hours Worked
- Rate Per Hour
- Plowing Round
- Today's Plowing Count
- Automatic Expense Calculation

👨‍🔧 Driver Management
- Add Drivers
- Driver Payment System
- Balance Tracking
- Payment History

📋 Records
- Plowing Records
- Driver Payment Records
- Total Paid
- Pending Balance
- Last Payment Date

---

🛠 Tech Stack

- Python
- FastAPI
- Streamlit
- SQLAlchemy
- MySQL
- Pandas
